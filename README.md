# Twitter-Sentiment-Analysis
## Overview
This project focuses on detecting hate speech in tweets using Natural Language Processing (NLP) and Machine Learning techniques.

The goal of the project is to classify tweets into two categories:
- **1 → Hate Speech (Racist/Sexist tweet)**
- **0 → Non-Hate Speech (Normal tweet)**

The model is trained on a labelled dataset containing **31,962 tweets**, where each tweet is assigned a label indicating whether it contains racist or sexist sentiment.

This project demonstrates how machine learning can be used to automatically identify harmful content on social media platforms.
